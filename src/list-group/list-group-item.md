## ListGroupItem

- 列表组件项

````jsx
  <ListGroupItem>111</ListGroupItem>
````

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| tag | 自定义标签 | string\|ReactNode | `div` |
| bstype | Bootstrap type | string | 无 |
| disabled | 是否禁用 | boolean | `false` |
| action | 改变组件hover时鼠标的指针样式 | boolean | `false` |
| active | 是否激活状态 | boolean | `false` |
