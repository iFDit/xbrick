## ModalDialog

- 显示Modal对话框

````jsx
  <ModalDialog>Dialog</ModalDialog>
````

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| tag | 自定义标签 | string\|ReactNode | `div` |
| align | 显示Dialog的位置, 有2个可选值： `start` \| `center` | string | |
| size | 设置Dialog宽度： `small` \| `middle` | `large` | `middle`|
| show | 显示或隐藏Dialog（内部） | boolean | `false` |
