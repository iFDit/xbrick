## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| tag | 自定义标签 | string\|ReactNode | `div` |
| size | 设置大小， 可选的值有：`small` 和 `large` | string | - |
| bstype | 设置颜色 | string | `primary` |
| split | 是否显示为分离样式 | boolean | `false` |
| navLink | 在NavLink组件使用 | boolean | `false` |
| button | 使用Button组件渲染 | boolean | `false` |
